# POLÍTICA DE PRIVACIDADE – APLICATIVO TOPA TUDO

A presente Política de Privacidade explica como o aplicativo Topa Tudo coleta, utiliza, armazena e protege as informações dos usuários que utilizam nossa plataforma de classificados locais.

## 1. INFORMAÇÕES QUE COLETAMOS
Para que o usuário possa publicar anúncios no Topa Tudo, coletamos as seguintes informações de forma direta e voluntária:
* **Dados de Cadastro do Anúncio:** Nome do anunciante, título do produto/serviço, preço estimado e categoria comercial.
* **Dados de Contato:** Número de telefone/WhatsApp do anunciante.
* **Mídias do Dispositivo:** Fotos (até 5 por anúncio) e vídeos demonstrativos (com limite de até 30 segundos) selecionados pelo próprio usuário a partir de sua galeria ou capturados pela câmera.

## 2. USO DAS PERMISSÕES DO ANDROID
O aplicativo solicita permissões nativas estritamente necessárias para o seu funcionamento:
* **Câmera (CAMERA):** Utilizada exclusivamente para capturar fotos ou gravar vídeos demonstrativos do produto no momento do cadastro.
* **Galeria/Armazenamento (READ_MEDIA_VISUAL_USER_SELECTED / READ_EXTERNAL_STORAGE):** Utilizada para permitir que o usuário selecione fotos e vídeos previamente salvos em seu Moto G54 ou outro dispositivo Android.

## 3. COMO OS DADOS SÃO UTILIZADOS E COMPARTILHADOS
Como o Topa Tudo é um mural público de classificados:
* O nome do anunciante e o número de telefone/WhatsApp inseridos serão exibidos abertamente no feed de anúncios para que potenciais compradores possam iniciar uma negociação direta via WhatsApp.
* Não compartilhamos, vendemos ou alugamos seus dados pessoais para empresas de publicidade ou terceiros. Todos os dados são enviados exclusivamente para o nosso servidor próprio.

## 4. ARMAZENAMENTO E SEGURANÇA
Todos os anúncios, dados de contato e arquivos de mídia são armazenados em um banco de dados seguro (SQLite) e em pastas diretivas protegidas em nosso servidor web próprio.

## 5. CONTROLE DO USUÁRIO E EXCLUSÃO DE DADOS
O usuário possui total autonomia sobre suas publicações. A qualquer momento, o anunciante pode acessar a seção "Meus Anúncios" dentro do aplicativo, informar o seu número de telefone cadastrado e efetuar a exclusão definitiva do seu anúncio. 
A exclusão remove imediatamente o registro do banco de dados e apaga fisicamente todas as fotos e vídeos associados dos arquivos do servidor.
